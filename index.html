lwip-fork

a fork of the light-weight IP stack, originally created by Adam Dunkels. LwIP is maintained at savannah and more info is available in the wiki

I created this fork to see if GitHub and its powerful integrations like Travis-CI, Coveralls and Gitter could help LwIP to speed up the release cycle a bit.

Consider this repository as an experiment for now!

See the original REAMDE file for more info.
